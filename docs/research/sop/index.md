# 个人 SOP

这里沉淀经过实际执行和复盘后，能够稳定复用的研究流程。

## 准入标准

一项经验进入个人 SOP 前，至少需要满足：

1. 已在真实研究任务中执行；
2. 输入、步骤、输出和失败条件清晰；
3. 关键结果能够复现；
4. 已记录人工判断点与工具能力边界；
5. 能说明适用场景和不适用场景。
